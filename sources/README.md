# Memory Game Afbeeldingen

Deze map bevat de afbeeldingen voor het memory game (Level 4).

## Benodigde afbeeldingen:
- `card1.jpg` - Eerste kaart afbeelding
- `card2.jpg` - Tweede kaart afbeelding
- `card3.jpg` - Derde kaart afbeelding
- `card4.jpg` - Vierde kaart afbeelding
- `card5.jpg` - Vijfde kaart afbeelding

Zet hier de afbeeldingen die je wilt gebruiken. De game maakt automatisch paren van deze afbeeldingen (elk wordt 2x gebruikt).

## Aanbevolen formaat:
- Resolutie: minimum 200x200 pixels
- Formaat: JPG of PNG
- Bestandsgrootte: onder 500KB per afbeelding
